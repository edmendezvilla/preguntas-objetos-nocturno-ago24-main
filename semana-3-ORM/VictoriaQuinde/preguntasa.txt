1. ¿Qué significa ORM en el contexto de desarrollo de software?

A. Object-Relational Mapping
B. Object-Remote Mapping
C. Object-Resource Mapping


ANSWER A porque su significado es Object-Relational Mapping.

2. ¿Cuál es el propósito principal de un ORM en el desarrollo de aplicaciones?
A. Convertir objetos en bases de datos relacionales
B. Convertir bases de datos relacionales en objetos
C. Convertir archivos JSON en bases de datos relacionales

ANSWER B porque el principal proposito de ORM es convertir bases de datos relacionales en objetos.

3. ¿Cuál de las siguientes afirmaciones es verdadera sobre el uso de un ORM?
A. Reduce la complejidad del código al eliminar la necesidad de escribir consultas SQL directamente.
B. Aumenta la complejidad del código al agregar una capa adicional de abstracción
C. No tiene ningún impacto en la eficiencia y el rendimiento de las consultas a la base de datos.

ANSWER A porque el uso de un orm reduce la complejidad del código al eliminar la necesidad de escribir consultas SQL directamente.
