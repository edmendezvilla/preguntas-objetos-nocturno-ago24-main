¿Cuál de las siguientes operaciones CRUD se utiliza para marcar un registro como "inactivo" sin eliminarlo físicamente de la base de datos?[4
]
A. Create
B. Read
C. Soft Delete
RESPUESTA: C

¿Qué tipo de operación CRUD se utiliza para realizar una copia exacta de un registro existente en la base de datos?[4
]
A. Create
B. Update
C. Duplicate
RESPUESTA: A

¿Cuál de las siguientes acciones NO se considera una operación CRUD en el contexto de una base de datos?[4]
A. Autenticar un usuario
B. Crear una nueva tabla
C. Actualizar la contraseña de un usuario
RESPUESTA: B
