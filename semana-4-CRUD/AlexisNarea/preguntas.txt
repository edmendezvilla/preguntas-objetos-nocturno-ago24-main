¿Qué decorador de Nest.js se utiliza para definir un método HTTP POST que crea una nueva entidad?[4]
A. @Get()
B. @Put()
C. @Post()
ANSWER: C

¿Qué patrón de diseño se utiliza para abstraer la interacción con la base de datos en Nest.js?[4]
A. Patrón Singleton
B. Patrón Repositorio
C. Patrón Data Mapper
ANSWER: B

¿Cuál de las siguientes prácticas NO es recomendable en el desarrollo CRUD con Nest.js?[4]
A. Utilizar servicios para encapsular la lógica de negocio relacionada con el acceso a datos.
B. Almacenar la lógica de negocio directamente en los controladores.
C. Definir métodos HTTP genéricos para las operaciones CRUD básicas.
ANSWER: B
